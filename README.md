# CSS Practice
This is my first attempt at replicating a game's menu style to practice learning and using CSS.

The website has a landing page and menu navigation to other pages that demonstrate gradient patterns, transitions, animations, layouts, scrolling frames, forms and interactive effects.

I spent two weeks on this unfinished project. After the lesson portion of David Gray's CSS course for beginners, I decided to substitute this in place of the final project. I felt he had taught me well enough to get started and search for anything else I needed to know along the way.

Most of my time was spent going through the documentation on MDN and exploring elements through the browser inspector, troubleshooting issues that either aren't well-covered or easily found on Stack Overflow and YouTube. I'd like to eventually record a full course on learning CSS through the making of this site, or a series of videos of my own about the complex issues I needed to resolve throughout the process, but will save that for when I have more time and experience.

In hindsight, I should have done the prescribed final project. I definitely learned a lot by doing this on my own, maybe built some of that "character" people are always talking about, but I was over zealous and could have learned more in less time if I had followed along with the instructor. It would have been better to attempted this later with more knowledge and more tools than just pure CSS. Some knowledge about different frameworks and UI/UX design would have made some things a lot easier. Still, I'm proud of the progress I was able to make after only a couple weeks. This project remains incomplete, and the following still need to be implemented:
- media queries for different sized screens
- redo classes to follow BEM naming convention
- polish up sizing and spacing
- abstract more common properties, combine repeated properties, remove unused and redundant properties
- review html for accessibility

One day, I'll come back and do this properly. For now, I think the exercise has served its purpose. I'm satisfied with leaving it here and taking the next step to learn javascript.
